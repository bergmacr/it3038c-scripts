**How to check for free space using the freespace.ps1 script*

-Download the "freespace.ps1" file.

-Open up an instance of PowerShell ISE.

-Click "Open"

-Go to the specified file path that "freespace.ps1" was downloaded to.

-Open up the file.

-After the script is loaded up, hit run and view the output window as it should have a small table with data about your PC's drives'.

-That is it! Enjoy.
